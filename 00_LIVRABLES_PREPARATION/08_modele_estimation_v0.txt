ÉTAPE 4 : INTERVALLE D'INCERTITUDE
================================================================================

Principe : Fournir [Bas, Probable, Haut]

Bas = Effort_Final × (1 - u)
Probable = Effort_Final
Haut = Effort_Final × (1 + u)

Facteur d'incertitude u :
-------------------------

u_base = 0.25 (par défaut, ±25%)

Ajustements :
• Si données d'entrée floues → +0.1
• Si type de projet rare (< 3 cas dataset) → +0.15
• Si nombreux facteurs Kinshasa élevés (≥3 facteurs niveau 4-5) → +0.2
• Si équipe très junior (niveau 4-5) → +0.15

u_final = u_base + ajustements
u_final = min(u_final, 0.8)  [plafond à 80%]


Exemple :
Effort_Final = 50 H/J
u = 0.25 + 0.1 (données floues) + 0.2 (Kinshasa difficile) = 0.55

Bas = 50 × (1 - 0.55) = 22.5 H/J
Probable = 50 H/J
Haut = 50 × (1 + 0.55) = 77.5 H/J
